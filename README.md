# Statistical Analysis of Runs in Behind

This repo contains code and reports for an analysis of how runs in behind influence possessions leading to shots in soccer.

The data is collected from SkillCorner Open Source Data https://github.com/SkillCorner/opendata

Processed and saved via code in `modules/get_data.py`

Analyzed in `run_analysis.ipynb`

View https://zcochran27.github.io/run_analysis/ for results
